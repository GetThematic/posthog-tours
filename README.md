# PostHog Tours

A TypeScript package for creating guided product tours in PostHog.

## Installation

```bash
npm install posthog-tours
```

or

```bash
yarn add posthog-tours
```

## Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the package:
   ```bash
   npm run build
   ```
4. Run tests:
   ```bash
   npm test
   ```
